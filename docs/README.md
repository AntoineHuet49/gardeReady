## 🏗️ Architecture

Le projet est organisé en trois composants principaux :
- **Backend** (`/Sources/api`) - API Node.js/Express/TypeScript
- **Frontend** (`/Sources/client`) - Application React/TypeScript
- **Database** (`/Sources/database`) - Scripts PostgreSQL
